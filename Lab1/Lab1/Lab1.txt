Функция ИзУглаУзнатьСторону(УголВГрадусах, Радиус)
    Дробное число ПИ = arccos(-1.0)

    Дробное число УголВРадианах = УголВРадианах * ПИ / 180
    Дробное число Сторона = 2 * Радиус * sin(УголВРадианах)

    вернуть Сторона

Начало
Дробные числа: a = 1.0, b = 1.0, c = 1.0, R = 0.0, A, B, C

Пока а + b + c != 180 выполнить:
    a = ввод (сторона а)
    если ошибка ввода или а <= 0, то:
        в начало цикла

    b = ввод (сторона b)
    если ошибка ввода или b <= 0, то:
        в начало цикла

    c = ввод (сторона c)
    если ошибка ввода или c <= 0, то:
        в начало цикла

Пока R <= 0 выполнить:
    R = ввод (радиус описанной окружности)
    если ошибка ввода, то:
        в начало цикла

A = ИзУглаУзнатьСторону(а, R)
B = ИзУглаУзнатьСторону(b, R)
C = ИзУглаУзнатьСторону(c, R)

Вывод: 1 сторона = А 
Вывод: 2 сторона = B
Вывод: 3 сторона = С
